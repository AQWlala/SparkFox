# sparkfox-chat

SparkFox 对话引擎（BaiLongma 对话模式清洁室重写）。

## Features

- liveEl streaming bubble
- Double dedup (ID + content + TTL 2min)
- Activation warmup lock
- Auto-grow input
- Push-to-talk (hold space)
- Multi-channel labels
