# sparkfox-monitor

SparkFox 监视数据（OpenAkita TokenStatsView 清洁室重写）。

## Features

- 6 periods (1d/3d/1w/1m/6m/1y)
- 5 dimension stats (total/input/output/requests/cost)
- Group by endpoint / operation_type
- Timeline (hour/day)
- Sessions list
- Usage records
- Extended: CRDT sync / E2EE health / 6-layer memory health
